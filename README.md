# Πρόβλεψη Καρδιακής Νόσου με Apache Spark (PySpark)
Αυτό το αποθετήριο περιέχει τον κώδικα για την Εξαμηνιαία Εργασία του μαθήματος **"Πλατφόρμες και Αρχιτεκτονικές Νέφους"**. 

## Στόχος
Η ανάπτυξη ενός κατανεμημένου συστήματος Μηχανικής Μάθησης για την πρόβλεψη καρδιακής νόσου, αξιοποιώντας πραγματικά ιατρικά δεδομένα. Υλοποιήθηκαν και συγκρίθηκαν δύο αλγόριθμοι ταξινόμησης για την αξιολόγηση της απόδοσής τους:
1. **Logistic Regression**
2. **Random Forest**

## Τεχνολογίες & Εργαλεία
* **Γλώσσα Προγραμματισμού:** Python
* **Κατανεμημένη Επεξεργασία:** Apache Spark (PySpark)
* **Machine Learning Library:** Spark MLlib
* **Ανάλυση & Οπτικοποίηση:** Pandas, Matplotlib, Seaborn

## Περιεχόμενα Αποθετηρίου
* `HeartDiseaseClassification.ipynb`: Το Jupyter Notebook (Google Colab) με όλο τον κώδικα (Data Preprocessing, Pipelines, Model Training, Evaluation).
* `heart.csv`: Το πραγματικό σύνολο δεδομένων που χρησιμοποιήθηκε για την εκπαίδευση των μοντέλων.

## Πώς να τρέξετε τον κώδικα
Ο πιο εύκολος τρόπος για να εκτελέσετε τον κώδικα χωρίς τοπικές εγκαταστάσεις είναι μέσω του Google Colab:
1. Ανεβάστε το αρχείο `.ipynb` στο Google Colab.
2. Ανεβάστε το αρχείο `heart.csv` στον προσωρινό χώρο αποθήκευσης (session storage) του Colab.
3. Εκτελέστε το πρώτο κελί για να εγκαταστήσετε το PySpark (`!pip install pyspark`).
4. Εκτελέστε τον υπόλοιπο κώδικα.
